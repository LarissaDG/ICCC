APDDv2-SynAE

A dataset for automated aesthetic analysis

Project Summary

APDDv2-SynAE (Aesthetics of Paintings and Drawings Dataset – Synthetic Aesthetic Evaluation) is an extension of APDDv2, proposed by (Jin et al., 2024), developed to investigate and automate the aesthetic analysis of AI-generated images. We sampled about 500 images from APDDv2, preserving the statistical relevance of the dataset. Each image was described using DeepSeek Janus, and from these textual descriptions, synthetic images were generated in the same style and content as the originals. The goal was to reproduce original dataset works from their descriptions alone, assuming that generated images should present similar aesthetic scores. Evaluation was performed using ArtCLIP, a model trained on the original dataset and proposed by APDDv2 authors, allowing comparison of aesthetic criteria between human-made and synthetic images. Presented at ICCC (International Conference on Computational Creativity) 2025 via the paper: Automatic Aesthetic Evaluation in Generative Image Models, APDDv2-SynAE expands possibilities for comparative studies between human and AI creations, contributing to computational art and computer vision advances.
